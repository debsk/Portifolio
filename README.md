# Portifolio - Debora Ferreira

Portfolio em Streamlit com projetos de QA, automacao e dados.

## Executar localmente

```bash
pip install -r requirements.txt
streamlit run portifolio.py
```

## Deploy no Streamlit Cloud

1. Suba este repositorio no GitHub.
2. Acesse https://share.streamlit.io
3. Clique em **New app**.
4. Selecione o repositorio `debsk/Portifolio`.
5. Em **Main file path**, informe `portifolio.py`.
6. Clique em **Deploy**.

## Arquivos principais

- `portifolio.py`
- `image.png`
- `requirements.txt`
